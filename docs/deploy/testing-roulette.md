# 🎰 Тестирование AI Рулетки

## Что изменилось:

### 1. ✅ Изменены кнопки после решения проблемы

Для **Предпринимателя** и **Стартапа** после получения решения теперь показывается только одна кнопка:
- **🎰 Крутить AI рулетку**

### 2. ✅ Автоматическое уведомление о выигрыше

После того как пользователь крутит рулетку и выигрывает приз, ему **автоматически приходит сообщение** от бота:

```
🎉 Поздравляем!

Вы выиграли скидку **[СУММА] ₽** на услуги нашей компании!

💰 Эта сумма будет вычтена из стоимости разработки вашего проекта.

📞 Свяжитесь с нами, чтобы использовать скидку:
• Сайт: [ваш сайт]
• Email: [ваш email]
• Телефон: [ваш телефон]

Спасибо за участие! 🚀
```

### 3. ✅ Сумма скидки сохраняется в БД

Приз сохраняется в таблице `roulette_spins` с привязкой к `telegram_id` пользователя.

## 📝 Как протестировать:

### Шаг 1: Начните диалог с ботом
```
/start
```

### Шаг 2: Выберите роль
- Выберите **"👔 Я предприниматель"** или **"🚀 Я стартапер"**

### Шаг 3: Ответьте на вопросы
- Ответьте на все вопросы квиза
- Поделитесь контактом или введите его вручную

### Шаг 4: Получите решение
После получения решения вы увидите сообщение с проблемой и решением, и **одну кнопку**:

```
🎰 Крутить AI рулетку
```

### Шаг 5: Нажмите на кнопку
- Откроется мини-приложение с рулеткой
- Крутите рулетку

### Шаг 6: Получите уведомление
Сразу после спина вы получите сообщение от бота с информацией о выигрыше!
