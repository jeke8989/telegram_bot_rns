# Документация для заказчика

Эта папка содержит документацию, написанную простым языком для людей, которые будут пользоваться продуктом. Здесь нет технических терминов и программного кода.

## Содержание

- [Как запустить бота](how-to-start.md) — пошаговая инструкция по запуску
- [Как пользоваться ботом](how-to-use.md) — что умеет бот и как с ним работать

## Правила написания документации в этой папке

1. Пишем на русском языке
2. Никаких IT-терминов без объяснения
3. Каждый шаг — с картинкой или скриншотом (при наличии)
4. Формат: «Сделайте это → получите это»
5. Целевая аудитория — человек, который никогда не работал с Telegram-ботами
