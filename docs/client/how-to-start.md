# Как запустить бота

## Что вам понадобится

- Компьютер с доступом в интернет
- Установленная программа Docker (ваш разработчик поможет с установкой)

## Пошаговая инструкция

### Шаг 1. Откройте терминал

- На Mac: найдите программу «Терминал» через поиск Spotlight (нажмите Cmd + Пробел и введите «Терминал»)
- На Windows: нажмите правой кнопкой на кнопку «Пуск» и выберите «PowerShell»

### Шаг 2. Перейдите в папку проекта

Введите команду (путь к папке вам даст разработчик):

```
cd путь/к/папке/проекта
```

### Шаг 3. Запустите бота

Введите команду:

```
docker-compose up -d
```

Подождите 1-2 минуты. Бот запустится автоматически.

### Шаг 4. Проверьте

Откройте Telegram, найдите вашего бота и напишите ему `/start`. Если бот ответил — всё работает!

## Как остановить бота

Введите в терминале:

```
docker-compose down
```

## Что-то не работает?

Свяжитесь с разработчиком и отправьте ему результат команды:

```
docker-compose logs
```
