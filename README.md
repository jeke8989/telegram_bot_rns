# Neuro-Connector Bot v3

**Neuro-Connector** — это многоуровневый Telegram-бот для нетворкинга, разработанный для конференций и мероприятий. Он помогает быстро собирать контакты, классифицировать аудиторию и находить точки для потенциального сотрудничества.

![Welcome Banner](assets/welcome_banner.png)

## 🚀 Ключевые особенности

- **Многоуровневая точка входа**: Бот определяет роль пользователя (предприниматель, стартапер, специалист, исследователь) и адаптирует сценарий.
- **Адаптивные сценарии**: 4 разных пути для каждого типа пользователя, чтобы собрать релевантную информацию.
- **AI-анализ**: Использует OpenRouter API (`gpt-4o`) для генерации персонализированных решений, рекомендаций и приветствий.
- **🎙️ Транскрибация аудио**: Поддержка голосовых сообщений! Пользователи могут отвечать голосом на любой вопрос бота. Используется Whisper API (OpenAI).
- **📇 Сканирование визиток**: AI Vision автоматически распознает все данные с визитной карточки (имя, должность, компания, телефон, email, сайт, адрес) и сохраняет в базу. Подробнее в [docs/features/business-card.md](docs/features/business-card.md).
- **🎰 AI Рулетка призов**: Telegram Mini App с интерактивной рулеткой для розыгрыша скидок на разработку от 5,000₽ до 30,000₽. Красивая анимация на Canvas, один спин на пользователя. Подробнее в документации проекта (docs/).
- **📢 Система рассылок**: Автоматическое сохранение всех пользователей бота в базу данных + скрипт для массовых рассылок. Подробнее в [docs/features/broadcast.md](docs/features/broadcast.md).
- **📲 Получение контактов**: Два способа - через кнопку "Поделиться контактом" в Telegram или ввод текстом.
- **◀️ Навигация назад**: На каждом шаге есть кнопка "Назад" для изменения ответов.
- **⚡ Inline кнопки**: Быстрые ответы на вопросы с готовыми вариантами.
- **Электронная визитка**: Автоматически отправляет красивую электронную визитку вашей компании.
- **Docker-контейнеризация**: Полностью готов к развертыванию с помощью Docker и Docker Compose.

## 🛠️ Технологический стек

- **Бот**: Python, `python-telegram-bot`
- **Mini App**: Vanilla JavaScript, HTML5 Canvas, CSS3 Animations
- **Web Server**: aiohttp, aiohttp-cors
- **База данных**: PostgreSQL (asyncpg)
- **AI**: OpenRouter API (`gpt-4o`), OpenAI Whisper
- **Оркестрация**: Docker, Docker Compose

## ⚙️ Установка и запуск

### 1. Клонируйте или скачайте репозиторий

Сначала скачайте и распакуйте архив с проектом на свой компьютер.

### 2. Настройте переменные окружения

В корневой папке проекта найдите файл `.env.example` и переименуйте его в `.env`. Затем откройте `.env` и заполните необходимые значения:

```env
# Telegram Bot Configuration
TELEGRAM_BOT_TOKEN=8364761240:AAGwq6m_pC-fkrzba3g6rEl77pywJ_OmZq4

# OpenRouter API Configuration
OPENROUTER_API_KEY=your_openrouter_api_key_here  # <--- ВАШ КЛЮЧ OPENROUTER
OPENROUTER_MODEL=gpt-4o

# Database Configuration
DATABASE_URL=postgresql://neuro_user:neuro_password@postgres:5432/neuro_connector
POSTGRES_USER=neuro_user
POSTGRES_PASSWORD=neuro_password
POSTGRES_DB=neuro_connector

# ... остальные переменные можно оставить по умолчанию
```

- `TELEGRAM_BOT_TOKEN`: Токен вашего Telegram-бота (уже вставлен).
- `OPENROUTER_API_KEY`: **Обязательно** вставьте ваш API-ключ от [OpenRouter](https://openrouter.ai/).

### 3. Запустите приложение с помощью Docker Compose

Откройте терминал (или PowerShell/CMD в Windows) в корневой папке проекта и выполните команду:

```bash
docker-compose up --build -d
```

Эта команда соберет образы, создаст и запустит контейнеры с ботом и базой данных в фоновом режиме (`-d`).

### 4. Проверьте работу бота

Откройте Telegram и найдите своего бота. Напишите ему команду `/start`. Бот должен ответить приветственным сообщением и предложить выбрать роль.

### 5. Остановка приложения

Чтобы остановить все контейнеры, выполните команду:

```bash
docker-compose down
```

## 🗄️ База данных

Проект использует PostgreSQL с несколькими таблицами:

### Основные таблицы

- **`users`** - ВСЕ пользователи бота (для рассылок)
  - Сохраняются автоматически при запуске команды `/start`
  - Используются для массовых рассылок сообщений
  - Отслеживается статус блокировки

- **`contacts`** - Пользователи, прошедшие опрос
  - Содержат полную информацию (телефон, email, роль)
  - Включают данные с визиток

- **`business_profiles`** - Профили предпринимателей
- **`startup_ideas`** - Профили стартаперов
- **`specialist_profiles`** - Профили специалистов
- **`roulette_spins`** - Результаты спинов рулетки

### Рассылка сообщений

Используйте скрипт `broadcast.py` для рассылок:

```bash
# Статистика пользователей
docker compose exec bot python broadcast.py --stats

# Отправка рассылки
docker compose exec bot python broadcast.py "Ваше сообщение"
```

Подробнее: [docs/features/broadcast.md](docs/features/broadcast.md)

## 🐛 Устранение неполадок

- **Бот не запускается**: Проверьте логи контейнера с помощью команды `docker-compose logs bot`. Убедитесь, что вы правильно заполнили `TELEGRAM_BOT_TOKEN` и `OPENROUTER_API_KEY` в файле `.env`.
- **Ошибка подключения к базе данных**: Убедитесь, что контейнер `postgres` запущен и здоров. Проверьте логи с помощью `docker-compose logs postgres`.
- **Изменения в коде не применяются**: Если вы внесли изменения в код, пересоберите образ с помощью `docker-compose up --build -d`.
